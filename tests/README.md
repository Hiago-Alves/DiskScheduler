# Escalonador de Disco com Árvore AVL

Projeto desenvolvido para a disciplina de Estrutura de Dados e Sistemas Operacionais.

## Objetivo

Simular algoritmos de escalonamento de disco utilizando uma árvore AVL para organizar as requisições por cilindro.

## Algoritmos

- FCFS
- SSTF
- SCAN
- C-SCAN
- C-LOOK
- Deadline
